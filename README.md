# Speakly

An AI-powered English shadowing and listening practice platform designed to help learners improve real-world English comprehension through audio-based lessons, bilingual subtitles, and sentence-by-sentence transcript interaction.

---

## Project Overview

Speakly is a lightweight English learning platform focused on:

- Listening practice
- Shadowing training
- Sentence-by-sentence transcript learning
- Audio replay and subtitle synchronization
- Real-life conversational English

The project is currently built as a frontend MVP using Bootstrap 5, HTML, CSS, and JavaScript, with future plans for a Java Spring Boot CMS backend and AI-powered subtitle generation tools.

---

## Features

### Current Features

- Responsive Bootstrap 5 UI
- Audio lesson player
- Sentence-by-sentence transcript display
- Bilingual subtitles (English + Chinese)
- Automatic subtitle highlighting during playback
- Click any sentence to replay audio from that timestamp
- Playback speed control (0.75x / 1x)
- Repeat current sentence mode
- Lesson detail page optimized for shadowing practice
